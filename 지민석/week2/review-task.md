### 준지도학습
- 설명: 
준지도학습은 지도학습과 비지도학습의 중간 형태로 일부 데이터는 라벨이 있고(소수) 나머지는 없는 상황(다수)에서 학습을 하기 위한 방식.
이는 라벨링 비용을 절감하면서 지도학습의 성능을 유지하기 위해 활용.
- 예시: 
음성 인식의 경우 일부 라벨이 있는 음성 데이터를 활용해 기본 모델을 학습하고 라벨이 없는 음성 데이터를 활용해 보강.
추천시스템에서 사용자의 일부 행동 데이터만 라벨링(좋아요, 구매한 제품 등)되어 있어 사용자 클릭과 머무른 시간 등의 라벨이 없는 데이터를 준지도학습을 활용해 보강.


### 오토인코더
- 설명:
입력 데이터를 인코딩하고, 디코딩하는 신경망 모델. 주로 비지도학습 기법으로 사용하고 데이터의 특징을 학습하는 것이 목적.
구조는 입력 데이터를 latent space로 압축하는 인코더, 중요한 정보만을 남겨 고차원 데이터를 저차원으로 변환한 latent space, 압축한 데이터를 원래의 데이터로 복원하는 디코더로 나뉨.

- 예시:
노이즈 제거 : 사진에서 노이즈를 제거하거나, 고화질로 변환하기 위함으로 
1. 원본 이미지에 노이즈 추가
2. 노이즈가 포함된 데이터를 오토인코더에 입력
3. 모델이 노이즈를 제거하여 깨끗한 이미지를 복원
의 순서로 작동하며, 이를 통해 저화질 사진을 고화질로 복원하거나 잡음 등을 제거하는 데에 사용 가능.
VAE,변분 오토인코더 : latent sapce에서 확률적 분포를 학습하여 새로운 데이터를 생성할 수 있게끔 하는 모델.
이미지나 음성, 영상 등을 VAE를 활용해 생성 가능.

### 분류 모델 평가 지표
- 설명:
TP(True Positive) : 실제 긍정인데 모델이 긍정으로 예측.
TN(True Negative) : 실제 부정인데 모델이 부정으로 예측.
FP(False Positive) : 실제 부정인데 모델이 긍정으로 예측.
FN(False Negative) : 실제 긍정인데 모델이 부정으로 예측.

정확도 : TP + TN / TP + TN + FP + FN
정밀도 : TP / TP + FP
재현율 : TP / TP + FN
F1-score : 2 * (정밀도*재현율)/(정밀도+재현율)

- 예시:
정확도 : 일반적인 균형 데이터셋에서 모델을 평가할 경우, 정확도가 중요
정밀도 : 잘못된 양성 예측을 줄여야 하는 경우 중요. 신용카드 사기 탐지 시스템에서 정상 거래를 사기 거래로 오인하면 안되는 경우.
재현율 : 잘못된 음성 예측을 줄여야하는 경우. 암 진단의 경우, 실제로 암인데 암이 아니라고 예측하면 치료에 큰 어려움.
F1-score : precision과 recall이 둘 다 중요한 경우. 재난 경보 시스템에서 화재 감지와 오탐의 균형이 필요.
