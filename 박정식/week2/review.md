## 주차 회고록

### 사전 과제

- 의사결정트리, 랜덤포레스트, 그래디언트 부스팅 결정 트리
- 준지도학습과 비지도학습
- 손실함수
- 모델 최적화

### 오늘의 KPT

Keep

- 각 비지도학습의 특징 분류 모델 평가 지표 

Problem

- 각 손실함수 수식이 등장한 배경은 어떻게 될까?
- 각 손실함수의 볼록성을 판단해볼 수 있을까?
- 각 비지도학습의 작동 원리가 무엇일까? 

Try

- 리뷰 과제

## 리뷰 과제 

- 수식 기반으로 손실함수 추가 탐구해보며 깊게 이해하기
- 비지도학습 방법별 구체적인 작동 원리 파악해보기
